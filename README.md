# devroast

Paste your code. Get roasted.

**devroast** gives your code a brutally honest AI review — scored from 0 to 10. No fluff, no sugar-coating. Just raw feedback on what's wrong (and occasionally what's right).

---

## What it does

- Paste any code snippet in any language
- Get an AI-powered review with a score from **0–10**
- See line-by-line diff annotations highlighting problems
- Check the **leaderboard** — the worst code ever submitted, ranked

---

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

---

## Stack

- **Next.js 16** · App Router · TypeScript
- **Tailwind CSS v4** · design tokens via `@theme`
- **Biome v2** · lint + format
- **Shiki** · syntax highlighting (theme: `vesper`)
- **Base UI** · accessible primitives (Toggle)
- **JetBrains Mono** · the only font that matters
